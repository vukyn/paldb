# Sphere Assembly Line

> A powered sphere line — split Pals into groups to out-produce the
> [[sphere-workbench|Sphere Workbench]]. Runs on electricity.

The mid-tier sphere station. It draws power (200 electricity/sec) and lets
several Pals work in parallel, so sphere output climbs well past the bench.
Drains a little sanity (SAN −0.15); defense 3.

## Notable crafts

All capture spheres, from the [[pal-sphere|Pal Sphere]] upward.

## Build

Unlocked at **Technology Lv 27**. Build workload: 30000 ([[handiwork|Handiwork]]).
Requires an electricity supply to operate.
