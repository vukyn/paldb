# Ancient Civilization Core

> Lõi từ công nghệ đã thất truyền từ lâu, chứa đựng kho tàng tri thức bị lãng

bậc cao nhất. Cực kỳ khan hiếm; cả lượt chơi bạn chỉ gom được vài cái.
