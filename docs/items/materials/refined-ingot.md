# Refined Ingot

> Required to create high-quality metal weapons and armor. Refined from Ore and
> Coal in a furnace.

## Obtain

Refine [[ore|Ore]] ×2 + [[coal|Coal]] ×2 in an
[[improved-furnace|Improved Furnace]] (or the [[ancient-furnace|Ancient Furnace]]).

## Use

High-quality metal weapons/armor and the [[ultra-sphere|Ultra Sphere]].
