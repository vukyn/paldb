# Arrow

> Basic ammunition for bows — the earliest ranged ammo.

Arrows for use with bows. Unlocked at Technology Lv 3 and craftable from the
[[primitive-workbench|Primitive Workbench]] onward. Cheap to make from
[[wood|Wood]] and [[stone|Stone]].
