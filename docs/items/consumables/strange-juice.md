# Strange Juice

> Juice that greatly improves mood when consumed. Drinking it sharply boosts work
> speed for a while, but sharply lowers sanity.

The stronger cousin of [[suspicious-juice|Suspicious Juice]]: a bigger **+400%
Work Speed** buff for a period, in exchange for a much faster [[sanity|Sanity]]
drain while it lasts. Use it to blitz heavy base work, but watch your Pals'
sanity crash afterward.
