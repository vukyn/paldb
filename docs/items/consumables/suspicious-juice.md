# Suspicious Juice

> Juice that improves mood when consumed. Drinking it boosts work speed for a
> while, but lowers sanity.

A mood-altering brew: drinking it lifts mood on the spot and grants a hefty
**+200% Work Speed** buff for a period, at the cost of steadily draining
[[sanity|Sanity]] while the effect lasts. Handy to rush a base task, risky if
overused.
