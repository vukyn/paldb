# Work Suitability

Every Pal has one or more **work suitabilities** — the jobs it can do at your
base (or inside a Palbox-assigned ranch/factory). When a Pal is assigned, it
automatically performs any task matching a suitability it has. Each suitability
a Pal has comes at a **level** (higher = works faster / unlocks higher-tier
structures); a Pal without a suitability cannot do that job at all.

The order Pals pick these jobs is [[work-priority|Work Priority]].

| ![](../assets/icons/work/lumbering.png){ .game-icon } | [Lumbering](work/lumbering.md) | Chops trees for wood. |
| ![](../assets/icons/work/mining.png){ .game-icon } | [Mining](work/mining.md) | Mines ore, stone and other rock nodes. |

| ![](../assets/icons/work/farming.png){ .game-icon } | [Farming](work/farming.md) | Produces ranch goods (eggs, wool, milk, etc.) when on a ranch. |

!!! note "Element affinity"
    A Pal's element often correlates with its suitabilities (Fire Pals tend to
    have Kindling, Water Pals Watering, etc.) but it is **not** a strict rule —
    always check the individual Pal. See [[elements|Elements]].
