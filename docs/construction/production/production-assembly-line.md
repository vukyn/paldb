# Production Assembly Line

> A powered assembly line — splitting Pals into groups lets it out-produce the
> workbenches. Runs on electricity.

The late-game [[handiwork|Handiwork]] station. Unlike the bench-tier stations it
draws power (200 electricity/sec) and lets several Pals work in parallel, so
throughput climbs sharply. Drains a little sanity (SAN −0.11); defense 3.

## Notable crafts

Makes [[gunpowder|Gunpowder]], [[cryogenic-coolant|Cryogenic Coolant]], and
other mid/late materials in bulk.

## Build

Unlocked at **Technology Lv 29**. Build workload: 10000 ([[handiwork|Handiwork]]).
Requires an electricity supply to operate.
