# Ancient Workbench

> A Pal-only bench that can craft any item in one place — the player can't use
> it. Runs at plain-workbench speed, but frees your hands entirely.

A fully automated station: it makes any recipe from a single bench, so a
well-staffed base needs no other crafting station. It cannot be operated by the
player and runs at the same speed as a basic bench — its value is automation, not
speed. Because of the advanced gear it demands Pals with **high-level
[[handiwork|Handiwork]]** *and* **[[medicine-production|Medicine Production]]**
suitability. Draws 2000 electricity/sec; drains a little sanity (SAN −0.11);
defense 4.

## Notable crafts

Any item — including [[gunpowder|Gunpowder]], spheres, and end-game gear.

## Build

Unlocked at **Technology Lv 67**. Build workload: 1000000 ([[handiwork|Handiwork]]).
Requires an electricity supply to operate.
