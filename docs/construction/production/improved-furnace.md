# Improved Furnace

> A better smelter — refines [[refined-ingot|Refined Ingot]]. Higher quality
> than the primitive furnace, but still slow. Needs a Fire Pal for the flame.

The mid-tier furnace, an upgrade over the
[[primitive-furnace|Primitive Furnace]] that unlocks
[[refined-ingot|Refined Ingot]]. Quality is up but throughput is still modest.
Needs the [[kindling|Kindling]] suitability. Drains a little sanity (SAN −0.15);
defense 3.

## Refines

- [[refined-ingot|Refined Ingot]] — from [[ore|Ore]] + [[coal|Coal]].

## Build

Unlocked at **Technology Lv 34**. Build workload: 30000 ([[handiwork|Handiwork]]).
