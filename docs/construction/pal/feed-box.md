# Feed Box

> A food storage box your working Pals eat from — keep it stocked to keep them happy.

A box for storing Pal food. Pals can feed from it when hungry. Make sure to keep
it well-stocked to keep your Pals happy — an empty Feed Box is the most common
cause of hunger loss and falling [Sanity](../../mechanics/sanity.md) at a base.

## Use

Place one at each base and fill it with food (berries, cooked dishes, etc.).
Working Pals automatically eat from it when their hunger drops. An unstocked box
lets Pals go hungry, which drains their [[sanity|Sanity]] over time. Defense 2.

## Build

Unlocked at **Technology Lv 4**. Build workload: 1000 ([[handiwork|Handiwork]]).
