# Ancient Pal Bed

> The ultimate Pal bed — highest healing and Sanity recovery of any bed.

An ultimate Pal bed made from an ancient recovery pod with a soft mattress. It
offers one of the highest SAN recovery rates of all Pal beds, quickly relieving
fatigue. It is the same size as a [Large Pal Bed](large-pal-bed.md), so large
Pals can rest comfortably.

## Rest

Restores **10 HP** per tick and recovers **0.5** [Sanity](../../mechanics/sanity.md)
(SAN Rate) — the best values of any Pal bed, roughly 3× the Large Pal Bed's
Sanity rate. Defense 4.

## Build

Unlocked at **Technology Lv 73**. Build workload: 100000 ([[handiwork|Handiwork]]).
