# Straw Pal Bed

> The cheapest Pal bed — where captured Pals rest and recover at night.

A straw bed for Pals. Pals rest on it when injured or to sleep at night. It
isn't soft, but it's better than nothing — the entry-level option before the
[Fluffy Pal Bed](fluffy-pal-bed.md).

## Rest

Restores **1.5 HP** per tick and recovers **0.05** [Sanity](../../mechanics/sanity.md)
(SAN Rate). The lowest values of any Pal bed. Defense 2.

## Build

Unlocked at **Technology Lv 3**. Build workload: 2000 ([[handiwork|Handiwork]]).
