# Egg Incubator

> Used for incubating a Pal Egg. Leave an egg in it and it hatches automatically
> after some time.

A base structure central to [[breeding|breeding]]: place any Pal Egg inside and
it hatches on its own over time — no worker needed to run it. Defense 3.

## Use

Drop an egg (e.g. [[common-egg|Common Egg]], [[rocky-egg|Rocky Egg]],
[[verdant-egg|Verdant Egg]], [[huge-verdant-egg|Huge Verdant Egg]],
[[dark-egg|Dark Egg]]) into the incubator and wait;
it hatches into the corresponding Pal. See [[breeding|breeding]] for how bred
eggs are produced.

## Build

Unlocked at **Technology Lv 10**. Build workload: 5000 ([[handiwork|Handiwork]]).
