# Flower Bed

> A decorative base structure that boosts your base's gathering speed by +20%.

Place in a base to improve gathering speed. Placing more than one has no
additional effect — the **+20%** bonus does not stack.

## Use

A base buff structure. While built in a base it raises gathering speed by
**+20%**. Only one instance is effective per base. Defense 3.

## Build

Unlocked at **Technology Lv 24**. Build workload: 32000 ([[handiwork|Handiwork]]).
